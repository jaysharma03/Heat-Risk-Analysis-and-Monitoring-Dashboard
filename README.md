# Environmental Heat Monitoring Dashboard

## 📌 Project Overview
The Environmental Heat Monitoring Dashboard is a Power BI-based analytics project designed to monitor environmental temperature, humidity, and effective heat conditions. The dashboard helps identify overheating and burn-risk situations using interactive visualizations and data analysis techniques.

---

## 🎯 Business Problem
Industries and workplaces face risks related to excessive environmental heat, overheating, and fire hazards. Manual monitoring of temperature and humidity is inefficient and may lead to delayed decision-making.

This project provides a smart monitoring solution to:
- Track environmental conditions
- Analyze effective heat generation
- Identify burn-risk situations
- Support safety and operational decisions

---

## 🚀 Features
- Interactive Power BI dashboard
- KPI cards for temperature, humidity, and burn percentage
- Heat trend analysis
- Burn-risk monitoring
- Humidity group analysis
- Dynamic filtering and visualization

## 📊 Dataset Columns
| Column Name | Description |
|---|---|
| Date | Monitoring date |
| Temperature_C | Temperature in Celsius |
| Humidity_% | Humidity percentage |
| Lens_Factor | Heat amplification factor |
| Effective_Heat | Calculated heat intensity |
| Burn_Paper | Burn risk indicator |
| Day Name | Day of week |
| Day Number | Numeric day value |
| Humidity Group | Humidity category |

---

## 📈 Dashboard Insights
- Higher temperatures increase effective heat
- Low humidity creates higher burn risk
- Effective heat peaks on high-temperature days
- Burn conditions are more common in low humidity ranges

---

## 💡 Business Recommendations
- Implement real-time heat monitoring
- Configure automated heat alerts
- Improve cooling and ventilation systems
- Use predictive analytics for risk forecasting
- Maintain safe humidity levels
